# Vrindha Superstore Annual Report | Excel Dashboard

## Overview
An interactive annual sales report dashboard for Vrindha 
Superstore built using Microsoft Excel, analysing orders, 
revenue, customer demographics and sales channels.

## Dashboard Highlights
- Order vs Sales trend across all 12 months
- Men vs Women customer split (64% Women, 36% Men)
- Age vs Gender analysis (Adults dominate at 40.20%)
- Top 5 states by revenue (Maharashtra leads at 3.5M)
- Order Status breakdown (92% Delivered)
- Order Channel analysis (Amazon leads at 36%)

## Interactive Filters
- Category filter (Blouse, Bottom, Ethnic Dress, Kurta)
- Channel filter (Ajio, Amazon, Flipkart, Meesho)
- Month filter (Jan to Dec)

## Key Insights
- Women are the dominant customer segment at 64%
- Amazon is the top sales channel with 36% share
- Uttar Pradesh and Maharashtra are top revenue states
- 92% of orders are successfully delivered
- Adult age group drives maximum sales at 40.20%

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Graphs
- Interactive Slicers and Filters
- Data Cleaning and Transformation
